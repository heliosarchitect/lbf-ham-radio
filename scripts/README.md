# Scripts Directory

This directory contains automation scripts for the lbf-ham-radio project.

## generate-changelog.py

Automatically generates `CHANGELOG.md` from git commit history using conventional commit format.

### Usage

```bash
# Generate changelog from current directory
python scripts/generate-changelog.py

# Generate changelog from specific repo path  
python scripts/generate-changelog.py /path/to/repo
```

### Commit Format

```
type(optional-scope): description

Examples:
feat: add frequency scanning mode
fix(cat): resolve serial timeout issue  
docs: update API reference
chore: bump version to v0.4.0
```

### Automation

The changelog is automatically generated by GitHub Actions:

- **On tag push** (`v*`): Generates changelog and publishes to PyPI
- **On main branch push**: Updates changelog and commits back to main
- **Manual trigger**: Available via workflow_dispatch

### Output Format

Generates a `CHANGELOG.md` file following [Keep a Changelog](https://keepachangelog.com/) format:

- Semantic versioning order (latest first)
- Commits grouped by type with emojis
- Links to repository and issues
- Short commit hashes for reference

### Dependencies  

- Python 3.9+
- Git repository with tags
- Conventional commit messages (recommended)
