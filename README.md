# LBF Ham Radio — Yaesu FT-991A Integration

AI-controlled amateur radio interface. CAT control via USB serial.

## Features (Planned)
- Full CAT command control (frequency, mode, power, PTT)
- Band monitoring and signal detection
- FT8/FT4 digital mode integration
- APRS position reporting
- Emergency beacon capability
- Voice synthesis over RF (TTS → modulator)

## Hardware
- **Radio**: Yaesu FT-991A (HF/VHF/UHF all-mode transceiver)
- **Interface**: USB CAT control (serial)
- **Computer**: Linux host with Python
